About Us

Welcome to Daraz Clone Project! This is a practice e-commerce website built using TailwindCSS.

Our aim is to help beginners understand how to build a clean and responsive e-commerce website step by step.

This project includes the following pages:

  • 🏠 Home Page
  • 🛒 Products Page
  • 🛍️ Cart Page
  • 💳 Checkout Page
  • ℹ️ About Page

🚀 With TailwindCSS, this website is fully responsive and works on desktop, tablet, and mobile devices.
